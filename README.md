🌟 Application : "Mes Tables de Multiplication"

# Test me here : https://cestymour.github.io/learn-table/

Cette application web interactive est conçue pour l'apprentissage et le perfectionnement des tables de multiplication par le jeu.

⚙️ Fonctionnalités principales
Personnalisation complète : Sélection des tables (de 1 à 12), du nombre de questions et de l'étendue des multiplicateurs.

4 Modes de jeu :

QCM : Sélection parmi 4 propositions.

Saisie libre : Réponse directe au clavier.

Speed QCM : Défi de rapidité (6 secondes par question).

Survie : Droit à seulement 3 erreurs.

Système de Progression : Chronomètre global optionnel, calcul de score avec bonus de combo (multiplicateur de points pour les séries de bonnes réponses) et barre de progression.

Analyse des résultats : Bilan détaillé avec score final, taux de réussite, meilleur combo et liste des erreurs commises pour favoriser la mémorisation corrective.

Interface Ludique : Feedback visuel immédiat (emojis, animations) et design adaptatif (compatible mobile).
